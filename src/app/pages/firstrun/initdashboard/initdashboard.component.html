Iniciar proyecto de Dashboard

Paso 1: Clona el repositorio en tu entorno de desarrollo local: 

	
git clone https://github.com/alejandromartincruz/classpip-dashboard.git

Paso 2: Instala las dependencias del proyecto: 

	
cd classpip-dashboard
npm install

Paso 3: Inicia Classpip-dashboard (por defecto en http://localhost:4200) con el siguiente comando: 

	
npm run start

Origen de los datos

Hay que asegurarse de que la aplicación de servicios esta funcionando antes de intentar usar la aplicación de mobile, ya que la aplicación de servicios es el origen y destino de los datos. De no ser así, al intentar iniciar sesión se obtendrá el error: ERROR: Ha ocurrido un error interno en el servidor. Por favor, contacta con tu administrador.

También hay que asegurar que este configurado el origen de los datos en localhost:3000. Para ello hay que dirigirse al archivo 'src/environments/environment.ts' y asegurarse de que el SERVER_URL apunta a localhost:3000 y no a https://dashboard.classpip.com basta con dejar las lineas como se muestra a continuación: 

		export const environment = {{'{'}}
  production: false,
  api: 'http://localhost:3000'
{{'}'}};
