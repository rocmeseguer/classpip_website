Iniciar proyecto de Onboarding

Paso 1: Clona el repositorio en tu entorno de desarrollo local: 

	
git clone https://github.com/alejandromartincruz/classpip-onboarding.git

Paso 2: Instala las dependencias del proyecto: 

	
cd classpip-onboarding
npm install

Paso 3: Inicia Classpip-onboarding (por defecto en http://localhost:4200) con el siguiente comando: 

	
ng serve --open
